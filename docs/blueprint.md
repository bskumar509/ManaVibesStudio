# **App Name**: Manavibes Studio Hub

## Core Features:

- Homepage Hero Section: Dynamic hero section with tagline and call-to-action buttons.
- Services Showcase: Display a detailed list of marketing, branding, and design services.
- Portfolio/Case Studies Carousel: Interactive carousel to showcase past projects and client success stories.
- Client Testimonials Slider: Rotating slider to display client feedback and ratings.
- Contact Form with Submission: Contact form for client inquiries.
- Footer with Social Links: Include links to social media profiles, quick links, and copyright information.
- Visual Design Assistant: AI tool to generate creative design suggestions, leveraging uploaded assets.

## Style Guidelines:

- Primary color: Energetic purple (#A050BE), inspired by creativity and digital innovation.
- Background color: Light, desaturated purple (#F1E5F4) for a modern backdrop.
- Accent color: Vibrant blue (#50B4BE) to highlight calls to action and key elements.
- Body and headline font: 'Poppins' (sans-serif) for a clean, professional look.
- Use clean and modern icons to represent services and key features.
- Responsive layout that adapts seamlessly to different screen sizes.
- Subtle animations and transitions for a dynamic and engaging user experience.
